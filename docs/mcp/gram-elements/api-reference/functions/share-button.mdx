# Function: ShareButton()

> **ShareButton**(`__namedParameters`): `Element`

A button component for sharing the current chat thread.
Copies a shareable URL to the clipboard when clicked.

## Parameters


<hr className="api-docs-separator" />

### \_\_namedParameters

[`ShareButtonProps`](/docs/mcp/gram-elements/api-reference/interfaces/share-button-props)

## Returns

`Element`

## Example

```tsx
import { ShareButton } from '@gram-ai/elements'
import { toast } from 'sonner'

function MyChat() {
  return (
    <ShareButton
      onShare={(result) => {
        if ('url' in result) {
          toast.success('Chat link copied!')
        } else {
          toast.error(result.error.message)
        }
      }}
    />
  )
}
```
