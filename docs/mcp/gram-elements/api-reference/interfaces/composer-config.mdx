# Interface: ComposerConfig

## Properties


<hr className="api-docs-separator" />

### placeholder?

> `optional` **placeholder**: `string`

The placeholder text for the composer input.

*Default*

```ts
'Send a message...'
```



<hr className="api-docs-separator" />

### attachments?

> `optional` **attachments**: `boolean` \| [`AttachmentsConfig`](/docs/mcp/gram-elements/api-reference/interfaces/attachments-config)

Configuration for file attachments in the composer.
Set to `false` to disable attachments entirely.
Set to `true` for default attachment behavior.
Or provide an object for fine-grained control.

*Default*

```ts
true
```



<hr className="api-docs-separator" />

### toolMentions?

> `optional` **toolMentions**: `boolean` \| [`ToolMentionsConfig`](/docs/mcp/gram-elements/api-reference/interfaces/tool-mentions-config)

Configuration for

*Tool*

mentions in the composer.
Set to `false` to disable, `true` for defaults, or an object for fine-grained control.

*Default*

```ts
true
```
