# Type Alias: Model

> **Model** = *typeof* [`MODELS`](/docs/mcp/gram-elements/api-reference/variables/models)\[`number`\]

The LLM model to use for the Elements library.

## Example

```ts
const config: ElementsConfig = {
  model: {
    defaultModel: 'openai/gpt-4o',
  },
}
```
